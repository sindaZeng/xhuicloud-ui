## 前言

`星辉云管理平台` 免费开源的后台模版。使用了最新的主流技术开发，开箱即用的前端解决方案，重新封装了多个刚刚组件，也可用于学习参考。

#### 技术栈支持
- **最新技术栈：**：使用 Vue3 等前端前沿技术开发
- **主题：**：可配置的主题
- **国际化：**：完善的国际化支持
- **权限：** 动态路由权限
- **组件：** 二次封装了多个常用的组件
- **社交登录：** 支持多种社交登录
- **Vite：** 分支vite即将支持

## 使用方法
### 安装依赖
```
npm install
```

### 项目编译
```
npm run build
```

### 生产环境运行
```
npm run prod
```

### 开发环境运行
```
npm run dev
```
